ZX81 REFERENCE

Graphics:

Shift + 1-8 for block graphics.
Shift + 9 for Graphics mode toggle.
Addr Function
BASIC I/O
$0010PRINT-CHARA=Char code. Destroys AF, BC, DE, HL.
$08F5PRINT-ATB=Row(0-21), C=Col(0-31).
$0A2ACLSClears D_FILE and resets system vars.
$0C0ESCROLLScroll up one line.
KEYBOARD
$02BBK-SCANRaw scan. Result in HL.
$07BDDECODEConverts raw scan to character in A.
GRAPHICS (QUARTER-PIXELS)
$0BB2PLOTY=B,X=C, T_ADDR < $9E
$0BB2UNPLOTY=B,X=C, T_ADDR > $9E
MODES & SYSTEM
$0F2BSLOWScreen visible during compute.
$0F23FASTCPU constant, screen goes black.
$0F35PAUSEPause te CPU

Note: Most calls destroy AF, BC, DE, and HL.
